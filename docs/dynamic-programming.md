# Dynamic Programming

## Key Concepts
- Solves overlapping subproblems
- Optimal substructure

## Common Problems
- Fibonacci Sequence
- Longest Common Subsequence
- 0/1 Knapsack

## Techniques
- Memoization (Top-Down)
- Tabulation (Bottom-Up)
