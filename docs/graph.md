# Graph

## Key Concepts
- Set of vertices connected by edges
- Directed/Undirected, Weighted/Unweighted

## Common Problems
- BFS, DFS traversal
- Detect Cycle
- Dijkstra's Algorithm

## Algorithms / Techniques
- BFS, DFS
- Union-Find
- Topological Sort
